# God Mode

{% stepper %}
{% step %}
[**God Mode (imortalidade) será concedido pela Staff apenas mediante clipe, comprovando que o player está dentro do limbo (injustiçado) nos seguintes casos:**](god-mode.md#god-mode-imortalidade-sera-concedido-pela-staff-apenas-mediante-clipe-comprovando-que-o-player-esta)

* Morte por NPC
* Queda de moto
* Batida de veículo
* Explosão do veículo

{% hint style="warning" %}
O God Mode não será mais concedido automaticamente pela Staff nos casos acima.
{% endhint %}
{% endstep %}

{% step %}
[**O que fazer em caso de morte:**](god-mode.md#o-que-fazer-em-caso-de-morte)

* O player deve:
  * Dar GG (Good Game) e aceitar a morte
  * Chamar um médico para reanimar (se tiver as condições)
* A Staff não concederá God Mode, a menos que o player comprove estar no limbo com clipe 30 segundos antes do ocorrido.

{% hint style="info" %}
Objetivo: Reduzir abusos do God Mode e aumentar a imersão no jogo.
{% endhint %}
